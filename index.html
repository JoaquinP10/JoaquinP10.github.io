My Work

Insector

En conjunto con la Universidad Católica, se realizó la adaptación de un robot móvil comercial basado en una tarjeta Raspberry Pi, para que opere con una tarjeta de desarrollo Tiva Launchpad. El producto final fue utilizado por los alumnos del curso de Sistemas Digitales, como parte del proyecto del curso. Adaptación de un robot móvil, originalmente diseñado con Raspberry Pi Pico W, para que funcione con una tarjeta de desarrollo Tiva Launchpad. El trabajo consistió en diseñar la tarjeta de conexiones necesaria y adaptar la programación para utilizar el Robot mediante la tarjeta de desarrollo. Asimismo, se realizaró la documentación necesaria para brindar información sobre la tarjeta, las conexiones y algoritmos utilizados.
